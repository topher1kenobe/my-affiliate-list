== Description ==

Creates a custom post type to hold affiliate links and list them on a single web page.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `my-affiliate-list` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create new or Edit a Post.
